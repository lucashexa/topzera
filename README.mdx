# ct-brazil-components

> Renders all basic components for call taker on call application

[![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

> To install the library run the following terminal command npm install git+https://github.com/gocarval/ct-brazil-components.git

## Usage

```jsx
import React, { Component } from 'react'

import CT from 'ct-brazil-components'

class Example extends Component {
  render () {
    return (
      <CT.MyComponent />
    )
  }
}
```
